# front_thechesscrew_fr

React + Vite Frontend for ia_thechesscrew_fr

## 🚀 Quick Start

### Setup
```bash
npm install
```

### Development
```bash
npm run dev
```
Frontend runs on `http://localhost:5173`

### Production Build
```bash
npm run build
```
Output in `dist/`

### Preview Production Build
```bash
npm run preview
```

## 🔌 Backend Integration

This frontend communicates with the backend API. Set the API URL in `.env`:

```
VITE_API_URL=http://localhost:3000
```

Available endpoints:
- `GET /api/health` - Health check
- `GET /api/hello` - Hello message
- `GET /api/data` - Sample data

## 📦 Technologies

- React 18+
- Vite
- CSS3 with gradients & animations
- CORS-enabled API calls
